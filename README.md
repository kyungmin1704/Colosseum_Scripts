# Colosseum_Scripts
Photon PUN을 활용한 멀티플레이 경험을 목표로 제작한 최대 8인 AOS 게임 팀 프로젝트입니다.
본 저장소는 저작권이 포함된 외부 에셋을 제외하고, 핵심 게임 로직을 담당하는 스크립트 위주로 구성된 Public Clone입니다.

## 플레이 과정 및 기능
### Auth
1. Firebase를 기반으로 이메일 형식 ID 가입 및 로그인

### Lobby
1. 로그아웃 버튼을 통해 ID/PW 미기입된 Auth로 복귀 가능
2. 커스터마이징 버튼을 통한 성별, 머리, 옷 등의 외형을 파츠별로 변경
3. 환경설정에서 해상도 조절 가능
4. 개인정보 패널에서 승/패 횟수 제공
5. 방을 생성하거나 다른 사람의 방을 선택해 진입
6. Quick Start를 누르면 즉시 방을 생성하거나 빈 방에 입장

### Room
1. 뒤로가기를 통해 Lobby로 복귀 가능
2. 중앙의 클래스(직업) 버튼들 중 원하는 것을 클릭하여 선택
3. 클래스 선택 시 활성화된 Ready 버튼 클릭이 가능해짐
4. 전원 Ready 및 양쪽 팀의 인원이 같으면 5초 후 게임 시작

### Ingame
1. 클래스별로 약공격, 강공격, 스킬 4개 지급
2. HP가 소진시 사망 및 관전, 공격 및 달리기에 소비되는 SP가 존재함
3. 이모티콘 버튼 4개를 통해서 약 3초동안 본인 머리 위에 모두에게 보이는 이모티콘 출력
4. 종료 시 로비로 복귀, 승/패 전적을 Lobby의 본인정보에 더함
