# Colosseum_Scripts
Photon PUN을 이용한 멀티플레이를 해보자는 취지로 제작된 최대 8인 AOS 게임제작 팀프로젝트입니다.
저작권이 포함된 외부 프롭 에셋 등을 제외하고 스크립트 위주로만 구성된 public 클론입니다.

## 개발 기간

## 플레이 과정 및 기능
1. Firebase를 기반으로 이메일 형식 ID 가입 및 로그인
2. 로비 진입 후 커스터마이징 버튼을 통한 성별, 머리, 옷 파츠별로 변경 가능
3. 환경설정에서 해상도 조절 가능
4. 방을 생성하거나 다른 사람의 방에 진입
5. 플레이하고 싶은 직업(클래스)을 선택하고 활성화된 ready 버튼 클릭
6. 전원 ready하면 5초 후 입장하여 직업별로 주어진 스킬로 HP(체력)/SP(스태미나자원)를 관리하며 한 팀이 전멸할때까지 대전 시작
7. 종료 시 로비로 복귀, 승/패 전적을 로비의 본인정보에 더함
